Thanks to FZF.vim, I could create it.

## Installation

I usually use vim-plug (https://github.com/junegunn/vim-plug)

Add this Line to the .vimrc
```vim
Plug 'dogfooter/FIP.vim'
```
Install plugin
```vim
:PlugInstall
```
